Procedure for Working Set

COVID-19 flow

  1. Reviewers submit comments.
  2. Reviewers respond to comments.
  3. Online tool manager exports Consolidated Comments.
  4. Chief Editor applies changes on the online tool based on Consolidated Comments.
  5. Online tool manager exports Change Log.
  6. Online tool manager generates new charts.

COVID-19 flow (old)

  1. Reviewers submit comments.
  2. Reviewers respond to comments.
  3. Online tool manager exports Consolidated Comments.
  4. Online tool manager exports Excel file for Chief Editor.
  5. Chief Editor updates Excel file based on Consolidated Comments.
  6. Online tool manager imports changes into tool.
  7. Online tool manager exports Change Log.
  8. Online tool manager generates new charts.

Normal flow

  1. Reviewers submit comments.
  2. Reviewers respond to comments.
  3. Online tool manager exports Consolidated Comments.
  4. IRG Meeting takes place and Discussion Records are recorded.
  5. Online tool manager applies preliminary changes on the online tool based on Discussion Record.
  6. Online tool manager exports Discussion Record with Change Log for Chief Editor.
  7. Chief Editor applies additional changes on the online tool based on the Discussion Record and Change Log.
  8. Online tool manager exports Change Log.
  9. Online tool manager generates new charts.
